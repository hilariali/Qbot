II. Scope and Precise Learning Outcomes (Compulsory Part, Module B: Computer System Fundamentals)
The questions generated must be derived exclusively from the following topics and their specific learning outcomes, which comprise the 20 hours allocated to Module B:
1. Topic B.a. Basic Machine Organisation (14 hours)
• LO 1.1: Explain the functions of hardware within a computer system, namely input and output devices, processing units (including CPU and GPU), bus system, and storage devices.
• LO 1.2: Explain the structure and functions of a CPU and its components. Students should know how CPU is measured in terms of frequency. Units such as microsecond, nanosecond and picosecond should also be introduced.
• LO 1.3: Outline the steps in the fetch-decode-execute cycle using a single processor, and describe the roles of and the interdependence among components, registers and buses in the machine cycle.
• LO 1.4: Describe the functions and characteristics of Random Access Memory (RAM), Read Only Memory (ROM) and memory cache. Realize the relationship among the size of the memory, the memory address, word length and the performance of the computer.
    ◦ Required knowledge: Students' understanding of main memory should be extended to current technologies in RAM and ROM (capacity and data access rate).
    ◦ Required knowledge: Meanings of units (terabytes, gigabytes, megabytes, and kilobytes) must be introduced. Distinction between computer prefixes (e.g., 1KB = 1024 Bytes) and Système International (SI) notation should be made.
• LO 1.5: Describe the features, advantages, disadvantages and applications of the input and output devices. Select and justify the use of appropriate devices for collecting and displaying information in a given context (Technical details on how each device operates are not required).
• LO 1.6: Describe the functional characteristics of storage devices in terms of random or sequential access, volatile or non-volatile, data transfer rate and storage capacity. (Examples: magnetic disk, optical disk, flash memory, magnetic tape, network storage).
• LO 1.7: Outline the latest developments in computer systems, including processor capabilities, primary memory technologies, secondary storage devices and data communications (Technical details are not required).
2. Topic B.b. System Software (6 hours)
• LO 2.1: Know the functions of system software and applications software, and the relationship between hardware, system software, applications software and users.
• LO 2.2: Outline the basic functions of an operating system and describe some common operating systems, and their differences and applications.
• LO 2.3: State the functions and needs of utility programs and driver programs. (Examples of utilities: data compressors, virus checkers, file managers, defragmentation software, and system monitoring software).
• LO 2.4: Distinguish the characteristics and applications of various modes of operation: batch processing, real-time processing, parallel processing, distributed processing, and virtualisation.
